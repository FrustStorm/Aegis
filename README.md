# Aegis (DevOps + DevSecOps) project


## Overview
This is a learning project to build an end-to-end DevOps + DevSecOps pipeline for a fake company called Aegis.

- infra/ Terraform snippet
- k8s/ Kubernetes manifest
- policies/ OPA/rego policies
- semgrep/ semgrep rules
